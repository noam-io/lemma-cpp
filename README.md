Noam Lemma - C++ Arm Linux
==========================

Lemma for embedded C++ applications.  

Currently tested on:
* Beaglebone Black
* Arduino Yun (also supports Serial with Arduino via LemmaApiSerial)


Included Dependencies
==========================
* Jansson (http://www.digip.org/jansson/)
